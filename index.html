Colors: Easier Than You Think

There are lots of ways to set, retrieve or manipulate colors through HTML and CSS. Luckily for us it as a very inuitive process, we'll go over some of them here. One of the key concepts we will touch on is the different ways to set colors, whether that be through RGB, HSL, Hexadecimal or default colors.

Red
background-color: red;

Red
background-color: rgb(255,0,0);

Red
background-color: hsl(0, 100%, 50%);

Red Hex (#FF0000)
with opacity set to 50%

As you can see, the last box has it's opacity, also called alpha, changed. This is essentially how see through an item is on the screen. It will blend in with other objects as well. In the picture below you can see two colors, red and blue, both with opacity set to 50% and overlapping.

CSS has a variety of built in colors that are available for use without using any other type of encoding. Take a second to browse over all the colors below, quite a bit eh?
